# Eliminar registros en SQL
> Para eliminar registros de una tabla utilizamos el comando **DELETE**
> debemos implementar un filtro con la palabra reservada **WHERE** seguida de una condición

> Sintaxis:  

    DELETE FROM nombreTabla  
      WHERE colPrimaryKey = valorID;


> Ejemplo práctico: 

    DELETE FROM clientes  
      WHERE idCliente = 25;
