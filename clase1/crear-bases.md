# Crear bases de datos en SQL
> Para crear una base de datos en SQL utilizamos el comando **CREATE DATABASE**

> Sintaxis:

    CREATE DATABASE nombre;

> En este curso nos vamos a referir a una base de datos llamada "introdb"

> ejemplo práctico:

    CREATE DATABASE introdb;

> Ejemplo avanzado

    CREATE DATABASE nombre   
      CHARACTER SET utf8mb4  
      COLLATE utf8mb4_0900_ai_ci;  


# Eliminar una base de datos en SQL
> Para eliminar una base de datos utilizamos el comando **DROP DATABASE**

> En la paleta de código escribimos:

    DROP DATABASE nombre;
